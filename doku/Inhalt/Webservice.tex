\section{UsersController}
\label{Webservice_UsersController}

Der \texttt{UsersController} stellt klassische Funktionen zur Benutzerverwaltung zur Verfügung.

\begin{itemize}
\item Register
\item Login
\item Logout
\item GetActiveUsers
\end{itemize}


\subsection{Register}
Registriert einen neuen Anwender in der Benutzerrolle \textit{Benutzer}.

Aufruf über HTTP Post:\\
\url{http://localhost:1129/api/Users/Register?userName=test&password=geheim}
